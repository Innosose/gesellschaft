# 게젤샤프트 (Gesellschaft)

바탕화면 위에 상주하는 AI 기반 오버레이 어시스턴트

---

## 개요

게젤샤프트는 단축키 한 번으로 화면 위에 나타나는 투명 오버레이 앱입니다.
중앙의 허브 버튼을 누르면 황금각 나선 형태로 도구들이 펼쳐지고,
AI가 현재 화면을 분석해 필요한 기능을 자동으로 추천합니다.

---

## 주요 기능

### 오버레이 인터페이스
- **글로벌 단축키** (기본: `Ctrl+Shift+G`) — OS 어디서든 즉시 토글
- **황금각 나선 메뉴** — 피보나치 수열 기반 배치, 22가지 도구
- **실시간 검색** — 메뉴 열린 상태에서 도구 이름 필터링
- **허브 이동** — 메뉴 전개 시 허브가 상단으로 이동해 중앙 시야 확보

### AI 어시스턴트
- **화면 분석** — 현재 바탕화면 스크린샷을 AI에 전송, 필요 도구 자동 추천
- **AI 채팅** — 각 도구 패널 내 AI 패널로 질문·작업 지원
- **멀티 프로바이더** — OpenAI / Anthropic / Ollama 선택 가능

### 내장 도구 (22종)
| 카테고리 | 도구 |
|---------|------|
| 파일 관리 | 고급 검색, 일괄 이름 변경, 폴더 비교 |
| 변환 도구 | CAD→PDF, PDF 병합/분할, 이미지 변환, Excel/CSV |
| 생산성 | 할일 목록, 업무 리마인더, 빠른 메모, 스니펫 관리, 이메일 템플릿, 클립보드 기록 |
| 계산기 | 부가세, 날짜 계산, 환율, 단위 변환 |
| 텍스트 | 한↔영 변환, 텍스트 도구 |
| 기타 | QR 코드 생성, 색상 피커, 이미지 OCR |

### 설정
- **테마** — 12가지 컬러 프리셋 + 커스텀 색상 피커
- **화면** — 허브 크기, 오버레이 불투명도, 나선 간격, 애니메이션 속도
- **단축키** — 글로벌 단축키 자유 지정
- **AI 연결** — 프로바이더·API 키·모델·시스템 프롬프트 설정

## 개발

```bash
npm install
npm run dev
```

## 빌드 (Windows 설치형)

Windows 환경에서:
```bash
npm run package
```

결과물: `dist/` 폴더에 `게젤샤프트 Setup.exe`

## 기술 스택

- **Electron 31** — 투명 프레임리스 오버레이 윈도우
- **React 18 + TypeScript** — 렌더러 UI
- **electron-vite** — 빌드 도구
- **electron-builder (NSIS)** — Windows 설치형 패키지
